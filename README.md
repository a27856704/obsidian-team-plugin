# Team Collaboration（Obsidian 插件 · 发布包）

本目录 **仅含编译产物**（如 `main.js`、`manifest.json`、`styles.css`），用于 **Obsidian 社区 / GitHub Release**。

**完整源码**（插件 + 服务端）在 **`obsidian-team/`**（与发布目录分离）。

**源码仓库参考**：  
<https://github.com/a27856704/obsidian-team-plugin-source>

## 安装

在 Obsidian：**设置 → 社区插件 → 浏览**，或手动将 Release 中的文件放入 `.obsidian/plugins/team-collaboration/`。

## 网络与隐私

详见源码仓库中的 README「网络使用与隐私说明」。
